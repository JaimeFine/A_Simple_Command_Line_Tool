# A_Simple_Command_Line_Tool

Based on the example projects from *The Rust Programming Language*,
licensed under CC BY-NC-SA 4.0 (https://creativecommons.org/licenses/by-nc-sa/4.0/).

In addition to its original function, I added a count_lines function that enhances the program by counting how many lines in a given text contain a specific query string. It filters the lines using a case-sensitive match and returns the total count as an integer. Depending on the result, it prints a contextual message to inform the user whether the query appears zero, one, or multiple times. This addition provides clearer feedback and complements the existing search functionality with a simple yet informative summary of matches.
